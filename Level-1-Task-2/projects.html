Spotify Clone
Developed a Spotify clone using HTML, CSS, and JavaScript, focusing on the frontend aspect. This music player application reads songs from a designated "songs" folder, which contains subfolders for various playlists. Each playlist includes songs and an info.json file that provides the playlist's name and description. Users can navigate through these playlists and play songs directly from the interface. While the project does not include backend functionality, it effectively simulates a music streaming experience with a user-friendly design and smooth navigation. The project is hosted on 000webhost.com for easy access and demonstration.
